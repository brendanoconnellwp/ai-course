## Who This Is For

This course is built for **developers who already build things** (especially WordPress developers) and want to integrate AI tooling into their real workflows — not toy demos.

You'll go from "I've heard of Claude Code" to confidently using AI agents to ship production code, automate repetitive tasks, and build on the Cloudflare platform.

## How It Works

Every lesson is a written guide with real commands you can run. No videos to pause and rewind — just copy, paste, and learn by doing. The entire course is open source markdown files, so you can even use Claude Code to navigate it.
